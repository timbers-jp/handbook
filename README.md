## 会社概要

東大を中心とするAIエンジニアやデータサイエンティストといった
若手技術者やその卵に30名以上のネットワークを持っており、
最新テクノロジーの社会実装と彼らの実践の場づくりに挑戦しております。

## 提供サービス

企業向けのデジタル推進の支援を行っております。

### Solution α
Google Workspaceを活用した低コストのデジタル推進

### Solution β
ローコードツールRetoolを活用したデジタル推進

### Solution γ
新領域でのAIの研究開発
